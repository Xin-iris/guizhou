# 贵州“村+”赛事展示网页

这是一个纯静态 HTML 展示页，适合在 VS Code 中直接预览或继续修改。

## 打开方式

1. 在 VS Code 中打开当前文件夹。
2. 直接打开 `index.html`。
3. 推荐使用 VS Code 的 Live Server 插件预览，交互和图片加载会更稳定。

## 文件说明

- `index.html`：页面结构与展示文案
- `styles.css`：视觉风格、响应式布局、动画样式
- `script.js`：滚动入场、数字增长、卡片切换、按钮交互
- `assets/`：从 PPT 中整理出的展示图片素材

## 页面内容

网页围绕贵州“村+”模式展开，包含首页、模式总览、榕江村超、台江村BA、村字头赛事矩阵、发展影响、灾后恢复、团队调研支撑等独立展示页。

其中“灾后重建榕江”页面已经作为重点强化，展示洪水侵袭后的救援、场地修复、商户复业和赛事重启路径。
